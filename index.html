OpenPifPafWebDemo Serverless

Experimental demo of a serverless version of OpenPifPaf. Only static assets are loaded from the web. All computation happens on the client side. Currently only shows a representation of the fields without decoding the poses. Input image size is 129x97.

Only works in landscape mode. Needs access to a camera (use Safari on iOS).
